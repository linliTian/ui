# Getting Started
### Reusable React component library designed to create a consistent look and feel.
 
## Installation

Using npm or yarn.
```
npm install --save @rtkwlf/ui react react-dom react-is react-select @emotion/react @emotion/styled
```
or
```
yarn add @rtkwlf/ui react react-dom react-is
```
## Usage

Import the components you would like to use in your React Application:

`import { Button } from @rtkwlf/ui;`

Then use them in a component:
```
const example = () => (
  <Button>Click Me!</Button>
);
```
