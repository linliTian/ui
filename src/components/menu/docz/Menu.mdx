# Menu
Can be used for navigation around an application

## Simple

Simple Menu that can be used in a variety of places for navigation or listing options.

## Groups

Group items together using the `ItemGroup` component.

<Playground>
  <ThemeContainer>
    <div style={{background: "transparent", maxWidth: "300px" }}>
        <Menu onClick={(key) => console.log(key)}>
        <Menu.ItemGroup title="Group 1">
          <Menu.Item itemKey={'1-1'} disabled>
            Sub Item 1
          </Menu.Item>
          <Menu.Item itemKey={'1-2'}>Sub Item 2</Menu.Item>
        </Menu.ItemGroup>
        <Menu.ItemGroup title="Group 2">
          <Menu.Item itemKey={'2-1'}>Sub Item 2</Menu.Item>
          <Menu.Item itemKey={'2-2'}>Sub Item 3</Menu.Item>
        </Menu.ItemGroup>
      </Menu>
    </div>
  </ThemeContainer>
</Playground>
