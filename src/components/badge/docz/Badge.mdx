# Badge
### Also known as a pill or label. Often used to indicate status.

## Simple

> Note: you must manually set the color on any non-text `children` you supply (e.g. icons)

<Playground>
  <Badge leftChildren={<Check />}>v10.42</Badge>
  <Spacer />
  <Badge rightChildren={<Check />} backgroundColor="red">
    v10.42
  </Badge>
  <Spacer />
  <Badge
    leftChildren={<Check />}
    rightChildren={<Check />}
    backgroundColor="yellow"
  >
    v10.42
  </Badge>
</Playground>

## Icons Only

Can be used with icons only

<Playground>
  <Container>
      <Badge leftChildren={<Exclamation />} backgroundColor="gray">
        <InfoCircle />
      </Badge>
      <Spacer />
      <Badge backgroundColor="blue">
        <InfoCircle />
      </Badge>
    </Container>
</Playground>

## Text Only

Centers the child node if no `rightChildren` or `leftChildren` are specified

<Playground>
  <Container>
    <Badge backgroundColor="purple">Text Only</Badge>
    <Spacer />
    <Badge backgroundColor="orange">?</Badge>
  </Container>
</Playground>
